# JVTech Landing Page

> Premium Enterprise Landing Page with 3D Animations

## ✨ Features

- 🚀 **Next.js 14** (App Router)
- 🎨 **Tailwind CSS** (Custom Theme)
- 🌐 **Multi-Language** (ID/EN/ZH)
- 🧊 **3D Animations** (React Three Fiber)
- 💎 **Enterprise UI/UX**
- 📱 **Fully Responsive**
- ⚡ **Vercel Ready**

## 🛠 Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **3D**: Three.js, React Three Fiber, Drei
- **State**: Zustand
- **Icons**: Lucide React
- **Font**: Sora, DM Sans

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## 📂 Project Structure

```
src/
├── app/             # Next.js App Router
├── components/
│   ├── 3d/          # 3D Scenes
│   └── ui/          # UI Components
├── content/         # JSON Config (Editable)
│   ├── content.json # Site Content
│   └── ui.json      # UI Labels
└── store/           # State Management
```

## 🔧 Maintenance

### Update Content
Edit `src/content/content.json` to update text, services, stats, etc.

### Update UI Labels
Edit `src/content/ui.json` to update navigation, buttons, labels.
